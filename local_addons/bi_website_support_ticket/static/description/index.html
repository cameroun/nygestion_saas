Website Customer Helpdesk Support Ticket

Website customer helpdesk support ticket module is used for give customer an interface where he/she can send support ticket request and attach documents from website.Support ticket will be send by email to customer and admin.

Website Customer Helpdesk Support Ticket


Submit Support Request.

User can enter details about subject of support ticket, category, email, support team and other related details about support ticket.

User also attach any document for support ticket.

After completing filling form, user can submit ticket.


Thank You Message.

After submitting the information you will land up to Thank you page and an email will be send to admin after receiving support ticket from customer.


Support Ticket Email.




Support Tickets On My Account

You can see created support ticket of login user from my account.




Support Tickets List.

After clicking on Your Ticket from my account, You can see all support ticket of login user.


View Support Ticket.

For detail view of ticket you can click on any support ticket.




Send Message and Attachments From Website.

User can send message and attachments from here.
Admin or Technician can communitcate with customer using chatter.


Thank You Message.

After submitting message and attachment you will land up to Thank you page and message and attachment will be attached to support ticket.




Edit Support Ticket.

Click on Edit Ticket button to edit already created support ticket.
When you click on edit ticket button,It will redirect at backend side to edit ticket.


Support Ticket Tree View.

Here you can see created all support ticket from website.




Closed Support Tickets.

Closed support tickets will be display here.


Support Ticket Form View.

Here you can see form view of created support ticket.




Assign Support Tickets.

Admin can assign support ticket to related helpdesk team.
After assign support ticket,Stage will be change from New to Assign.



Escalate Team.

If you want to assign this ticket to higher level/parent team then you can use this button.
After using this button helpdesk team and team leader will be change as per parent team.
If you are in parent level then you can not escalate team.




Once you escalate team then level is also change.



Close Support Team.

Admin can close support ticket.
When you click on Set To Close button then stage will be change to closed and current date and time will be set as closed date.


Create Invoice.

When ticket is closed then after admin can create invoice.

You have 2 different options to create Invoice.

1. Create Invoice with Employee Timesheet.

By selecting invoice policy as Timesheet then you have also 2 option to create invoice.

1.1 - Using Employee Timesheet Cost.

1.2 - Using Manual Employee Cost.

If You select manual employee cost then you have to pass cost for creating invoice.




2. Create Invoice with Manual.

Click on task smart button to view list of support ticket tasks.




Reset Support Ticket.

Click on Set To Reset button to reset support ticket and stage will be set as New from Closed.


Support Ticket Report.

PDF report for support ticket.


Customer Rating Button.

After click on that button, It will redirect to rating page.
